안녕하세요, 부자타임입니다.

유튜브 초보자가 가장 긴장하는 순간은 "첫 영상 업로드 버튼"을 누르기 직전입니다.

> "혹시 설정 빠진 거 없을까?"

이 불안은 아주 정상입니다.  
그래서 이번 9편에서는 **첫 업로드 전 체크리스트 15개**를 초보자 기준으로 아주 쉽게 정리해드립니다.

---

## 유튜브 시작하기 연재 9편
# 첫 영상 업로드 전 체크리스트 15개 (초보자 스텝바이스텝)

---

## STEP 1) 업로드 전에 꼭 확인할 핵심 원칙

### 왜 필요한가?
첫 영상은 완벽보다 "실수 없이 게시"가 목표입니다.

### 실행 방법
- 핵심 1: 공개 전에 기본 설정 점검
- 핵심 2: 제목/썸네일/설명은 최소 완성
- 핵심 3: 업로드 후 24시간 데이터 기록

### 초보자 주의점
- 첫 영상부터 100점 욕심을 내면 업로드가 지연됩니다.
- 체크리스트로 안정적으로 1개 올리는 것이 더 중요합니다.

---

## STEP 2) 체크리스트 15개 (업로드 직전용)

아래 15개를 순서대로 체크하세요.

### A. 영상 파일 점검 (1~5)
1. [ ] 영상 해상도 1080p 확인
2. [ ] 소리(음량/잡음) 최종 확인
3. [ ] 영상 앞뒤 불필요 구간 컷 편집 완료
4. [ ] 오프닝/마무리 멘트 포함 확인
5. [ ] 파일명 정리 완료 (예: youtube-start-01.mp4)

### B. 메타 정보 점검 (6~10)
6. [ ] 제목 작성 완료
7. [ ] 설명란 작성 완료
8. [ ] 썸네일 업로드 완료
9. [ ] 카테고리/재생목록 설정 완료
10. [ ] 태그(핵심 5~10개) 입력 완료

### C. 공개 설정 점검 (11~15)
11. [ ] 공개 상태 확인 (일부공개/예약/공개)
12. [ ] 아동용 콘텐츠 여부 정확히 선택
13. [ ] 저작권 문제 요소(음원/이미지) 점검
14. [ ] 최종 미리보기(모바일/PC) 확인
15. [ ] 업로드 후 점검 메모 템플릿 준비

---

## STEP 3) 제목·설명·썸네일 최소 템플릿

### 왜 필요한가?
초보자는 매번 새로 쓰려다 시간이 오래 걸립니다.

### 실행 방법
아래 템플릿을 복붙해서 사용하세요.

#### 제목 템플릿
> "유튜브 초보 [주제]: [핵심 해결]"

예시:
- "유튜브 초보 첫 업로드: 실수 없이 올리는 체크리스트 15개"

#### 설명란 템플릿
```
안녕하세요, 부자타임입니다.
이번 영상은 유튜브를 처음 시작하는 분들을 위해 업로드 전 필수 체크리스트를 정리했습니다.

✅ 이런 분께 추천
- 첫 영상을 올리기 전에 불안한 분
- 설정 실수를 줄이고 싶은 분

다음 편: 첫 10개 영상 주제 뽑는 공식
```

#### 썸네일 문구 템플릿
- 1줄: `첫 업로드 전`
- 2줄: `체크리스트 15개`

### 초보자 주의점
- 제목을 너무 길게 쓰면 모바일에서 잘립니다.
- 설명란을 비워두면 검색 노출에 불리합니다.

---

## STEP 4) 업로드 직후 24시간 점검법

### 왜 필요한가?
업로드는 끝이 아니라 시작입니다. 첫 24시간 데이터를 보면 다음 영상 개선점이 바로 보입니다.

### 실행 방법
아래 3가지만 기록하세요.

1. 클릭률(CTR)
2. 평균 시청 지속 시간
3. 댓글/반응에서 나온 개선 포인트 1개

#### 복붙 메모 템플릿
```
[영상 제목]
- CTR:
- 평균 시청 지속 시간:
- 다음 영상 개선 1가지:
```

### 초보자 주의점
- 첫 영상 조회수가 낮아도 정상입니다.
- 1개 영상 데이터만으로 채널 방향을 바꾸지 마세요.

---

## 초보자 실수 TOP 5

1. 공개 상태 실수(미완성 영상 공개)
2. 썸네일 없이 업로드
3. 설명란/태그 생략
4. 저작권 요소 미점검
5. 업로드 후 데이터 기록 안 함
