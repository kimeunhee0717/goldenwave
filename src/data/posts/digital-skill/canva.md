# 캔바(Canva) 템플릿 직접 만드는 방법

---

## **1. 캔바 접속 및 설정**

### Step 1: 캔바 접속
- 주소: https://www.canva.com
- 로그인 (Google 계정으로 가능)

### Step 2: 크기 설정
1. 우측 상단 **"Create a design"** 버튼 클릭
2. **"Custom size"** 선택
3. 입력:
   - Width: **20 cm**
   - Height: **20 cm** (정방형 앨범용)
   - 또는 A4 (21cm x 29.7cm)
4. **"Create new design"** 클릭

---

## **2. 템플릿 A: 2장 배치 (반반)**

### 레이아웃 만들기

**Step 1: 배경 설정**
- 왼쪽 메뉴 → **"Elements"** 클릭
- **"Square"** 검색 → 사각형 선택
- 화면 가득 크기로 배치
- 색상: 흰색 (#FFFFFF) 또는 연한 베이지 (#F5F5DC)

**Step 2: 사진 프레임 2개 추가**
- **"Elements"** → **"Frame"** 검색
- **"Photo frame"** 중 사각형 선택
- 2개를 나란히 배치
- 각 프레임 크기: 가로 8cm x 세로 8cm
- 간격: 중앙에서 0.5cm 띄우기

**Step 3: 정렬**
- 두 프레임 모두 선택 (Shift + 클릭)
- 상단 **"Position"** 클릭
- **"Center horizontally"** 선택

**Step 4: 텍스트 공간 추가**
- **"Text"** 클릭 → **"Add a heading"**
- 아래쪽에 배치
- 예시: "2024.01.15  바다여행"
- 폰트: Noto Sans KR
- 크기: 14pt

**Step 5: 저장**
- 상단 **"File"** → **"Save"**
- 이름: **"앨범_2장배치_템플릿"**

---

## **3. 템플릿 B: 1대3 배치**

### 레이아웃 만들기

**Step 1: 큰 사진 프레임 (상단)**
- **"Frame"** → 큰 사각형 선택
- 크기: 가로 18cm x 세로 12cm
- 상단에 배치

**Step 2: 작은 사진 프레임 3개 (하단)**
- **"Frame"** → 작은 사각형 선택
- 각각 크기: 5.5cm x 5.5cm
- 3개를 나란히 하단에 배치
- 간격: 각각 0.5cm

**Step 3: 정렬**
- 3개 작은 프레임 선택 → **"Position"** → **"Distribute horizontally"**
- 전체 선택 → **"Center horizontally"**

**Step 4: 저장**
- 이름: **"앨범_1대3배치_템플릿"**

---

## **4. 템플릿 C: 폴리로이드 스타일**

### 레이아웃 만들기

**Step 1: 테두리 만들기**
- **"Elements"** → 사각형 선택
- 크기: 12cm x 14cm
- 색상: 흰색
- 테두리: **"Border"** 추가 (두께 10px, 색상: #E0E0E0)

**Step 2: 사진 프레임 (낸부)**
- **"Frame"** 추가
- 크기: 10cm x 10cm
- 테두리 안쪽 중앙에 배치

**Step 3: 캡션 공간**
- **"Text"** 추가
- 사진 아래에 배치
- 예시: "할머니 생신 축하"
- 폰트: 손글씨 스타일 (Hi Melody 또는 Gowun Batang)

**Step 4: 저장**
- 이름: **"앨범_폴리로이드_템플릿"**

---

## **5. 템플릿 D: 4그리드**

### 레이아웃 만들기

**Step 1: 사진 프레임 4개**
- **"Frame"** → 정사각형 선택
- 각 크기: 8cm x 8cm
- 2x2 배치

**Step 2: 배치**
```
[프레임1] [프레임2]
[프레임3] [프레임4]
```
- 간격: 가로/세로 모두 0.5cm
- 전체 중앙 정렬

**Step 3: 저장**
- 이름: **"앨범_4그리드_템플릿"**

---

## **6. 템플릿 재사용하기**

### 방법 1: 복사해서 사용
1. 완성된 템플릿 열기
2. 우측 상단 **"Duplicate"** 클릭
3. 새 파일에서 사진/텍스트만 교체

### 방법 2: 브랜드 키트에 저장 (유료 기능)
1. **"File"** → **"Brand Kit"** (왼쪽 메뉴)
2. **"Uploads"** → 템플릿 PNG 업로드
3. 다음부터 **"Brand Kit"**에서 꺼내쓰기

### 방법 3: 팀 템플릿으로 저장
1. **"File"** → **"Save as template"**
2. 팀 템플릿으로 저장
3. 가족들과 공유 가능

---

## **7. 사진 넣는 방법**

### Step 1: 사진 업로드
- 왼쪽 **"Uploads"** 클릭
- **"Upload files"** 버튼
- 컴퓨터에서 사진 선택

### Step 2: 프레임에 사진 넣기
- 업로드된 사진 클릭 → 드래그
- 프레임 위에 떨어뜨리기 (Drop)
- 자동으로 프레임 크기에 맞춰짐

### Step 3: 사진 조정
- 프레임 더블클릭
- 사진 위치 드래그로 조정
- 크기 조절 (마우스 휠 또는 핸들)
- 배경 클릭 → 완료

---

## **8. 공유 후 PDF 저장**

### Step 1: 공유하기
- 상단 **"Share"** 버튼 클릭
- **"Download"** 선택

### Step 2: 설정
- File type: **PDF Print** (인쇄용)
- 또는 **PNG** (이미지용)
- **"Download"** 클릭

### Step 3: 인쇄소 제출
- 다운받은 PDF를 USB에 저장
- 또는 이메일로 인쇄소에 전송
