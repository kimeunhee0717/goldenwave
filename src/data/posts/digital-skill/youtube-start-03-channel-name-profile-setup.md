안녕하세요, 부자타임입니다.

유튜브를 처음 시작하실 때 3편에서 가장 많이 막히는 부분이 바로 이것입니다.

> "채널명은 어떻게 정하지? 소개글은 뭘 써야 하지? 프로필 사진은 꼭 있어야 하나?"

오늘은 완전 초보자 기준으로, **채널명 / 소개글 / 프로필**을 한 번에 끝내는 방법을 단계별로 안내드리겠습니다.

---

## 유튜브 시작하기 연재 3편
# 유튜브 채널명·소개글·프로필 세팅 가이드 (초보자 스텝바이스텝)

---

## STEP 1) 채널명 방향 먼저 정하기

### 왜 필요한가?
채널명은 채널의 첫인상입니다. 이름이 모호하면 시청자가 채널 주제를 바로 이해하지 못합니다.

### 실행 방법
채널명을 만들 때 아래 3원칙을 먼저 적용하세요.

1. **누구를 위한 채널인지 보이게**
2. **무엇을 다루는지 보이게**
3. **너무 길지 않게**

권장 길이: 6~14자

#### 복붙 템플릿
- `[대상] + [주제]`
  - 예: "초보유튜브 길잡이"
- `[주제] + [도움 방식]`
  - 예: "유튜브 쉬운설명"
- `[브랜드명] + [핵심 키워드]`
  - 예: "부자타임 유튜브시작"

### 초보자 주의점
- 숫자/기호를 과도하게 넣지 마세요.
- 영어+한글을 무리하게 섞지 마세요.
- 오늘 유행어를 이름에 넣으면 나중에 촌스러워질 수 있습니다.

---

## STEP 2) 채널명 후보 5개 만들고 점수 매기기

### 왜 필요한가?
처음 떠오른 이름이 항상 좋은 이름은 아닙니다. 비교해야 더 좋은 선택이 됩니다.

### 실행 방법
후보 5개를 만들고 아래 기준으로 5점 만점 평가하세요.

- 이해도: 이름만 보고 주제가 보이는가?
- 기억도: 한 번 보면 기억나는가?
- 검색도: 검색할 때 헷갈리지 않는가?

#### 복붙 점수표
```
1) 채널명: __________
이해도(1~5) / 기억도(1~5) / 검색도(1~5) = 총점
```

총점이 가장 높은 이름 1개를 최종 선택하세요.

### 주의점
- 가족/지인 취향보다 "시청자가 이해하기 쉬운지"를 우선하세요.
- 너무 추상적인 감성 단어만 쓰지 마세요. (예: 빛, 꿈, 행복 등 단독 사용)

---

## STEP 3) 채널 소개글(About) 3줄 작성하기

### 왜 필요한가?
소개글은 구독 전 마지막 확인 포인트입니다. 이 채널이 나에게 맞는지 판단하는 문장입니다.

### 실행 방법
소개글은 아래 3줄 구조로 작성하면 쉽습니다.

1. 누구를 위한 채널인지
2. 어떤 내용을 다루는지
3. 어떤 결과를 돕는지

#### 복붙 템플릿 A
```
이 채널은 유튜브를 처음 시작하는 초보자를 위한 채널입니다.
채널 개설, 콘텐츠 기획, 촬영·편집, 업로드까지 차근차근 설명합니다.
처음 시작하는 분도 첫 영상부터 꾸준한 운영까지 가실 수 있게 돕겠습니다.
```

#### 복붙 템플릿 B (부자타임 스타일)
```
부자타임 유튜브 시작하기 시리즈입니다.
초보자가 막히는 부분을 쉬운 말로 단계별 정리해드립니다.
오늘 바로 따라할 수 있는 실전 방법만 안내합니다.
```

### 주의점
- 소개글에서 너무 많은 약속(수익화, 떡상, 자동화 완전정복)을 한 번에 하지 마세요.
- 어려운 전문 용어 남발은 초보자 이탈을 만듭니다.

---

## STEP 4) 프로필 사진/배너 최소 세팅하기

### 왜 필요한가?
프로필과 배너는 채널 신뢰도를 크게 좌우합니다. 초보 채널도 기본만 맞추면 훨씬 전문적으로 보입니다.

### 실행 방법
#### 1) 프로필 사진
- 얼굴 공개 가능: 밝은 배경 + 정면 사진
- 얼굴 비공개: 채널명 이니셜/아이콘으로 통일

#### 2) 배너 문구
아래 2줄이면 충분합니다.
- 메인 문구: "유튜브 시작하기, 초보자도 할 수 있습니다"
- 서브 문구: "채널 개설부터 첫 업로드까지 단계별 가이드"

#### 3) 색상 통일
- 메인 1색 + 보조 1색만 사용
- 예: 네이비 + 화이트 / 초록 + 화이트

### 주의점
- 배너에 글자를 너무 많이 넣지 마세요.
- 프로필/배너/썸네일 톤이 매번 다르면 브랜드 인지가 떨어집니다.

---

## STEP 5) 채널명·소개글·프로필 최종 점검

### 왜 필요한가?
세팅은 했는데 실제 방문자 입장에서 보면 어색한 경우가 많습니다.

### 실행 방법
아래 순서로 5분 점검하세요.

1. 내 채널을 처음 보는 사람처럼 홈 화면 열기
2. 5초 안에 "무슨 채널인지" 이해되는지 확인
3. 소개글 3줄 읽고 "누구를 위한 채널인지" 보이는지 확인
4. 프로필/배너 색감이 통일됐는지 확인
5. 핸드폰 화면에서도 글자가 잘 보이는지 확인

### 주의점
- PC 화면만 보고 끝내지 마세요. (모바일 확인 필수)
- 너무 완벽하게 만들려다 시작을 미루지 마세요.

---

## 초보자 실수 TOP 5

1. 채널명을 너무 어렵게 지음
2. 소개글을 비워둠
3. 배너 없이 운영 시작
4. 프로필·배너·썸네일 톤이 제각각
5. 본인 취향만 반영하고 시청자 이해도를 놓침

3. 프로필/배너 최소 세팅 후 모바일 화면에서 최종 점검
