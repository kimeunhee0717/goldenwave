# 조회수 정체 시 점검할 10가지 (초보자 스텝바이스텝)

## STEP 1) 먼저 "정체"를 데이터로 확인하기

### 왜 필요한가?
느낌만으로 정체라고 판단하면 잘못된 결론을 내릴 수 있습니다.

### 실행 방법
최근 5개 영상의 아래 항목을 비교하세요.

- 조회수 추이
- 클릭률(CTR)
- 평균 시청 지속 시간

#### 복붙 템플릿
```
[최근 5개 영상 점검]
- 영상1: 조회수 / CTR / 평균 시청
- 영상2: 조회수 / CTR / 평균 시청
- 영상3: 조회수 / CTR / 평균 시청
- 영상4: 조회수 / CTR / 평균 시청
- 영상5: 조회수 / CTR / 평균 시청
```

### 초보자 주의점
- 단일 영상으로 채널 전체를 판단하지 마세요.
- 최소 3~5개 영상 흐름을 같이 보셔야 정확합니다.

---

## STEP 2) 점검 10가지 (원인 진단 체크리스트)

아래 10가지를 순서대로 점검하세요.

### 1) 제목이 모호하지 않은가?
- 문제: 제목이 추상적이면 검색/클릭이 약해집니다.
- 해결: 대상+문제+해결 구조로 수정

### 2) 썸네일 문구가 너무 많지 않은가?
- 문제: 모바일에서 안 읽히면 클릭률 하락
- 해결: 3~6단어 핵심 문구만 유지

### 3) 제목-썸네일 메시지가 일치하는가?
- 문제: 불일치 시 낚시성 인식
- 해결: 같은 주제축으로 통일

### 4) 오프닝이 길지 않은가?
- 문제: 초반 이탈 증가
- 해결: 15~30초 내 문제/결과 제시

### 5) 본문이 너무 장황하지 않은가?
- 문제: 핵심이 늦게 나오면 중간 이탈
- 해결: 핵심 3포인트 구조로 재편

### 6) 영상 길이가 채널 단계에 맞는가?
- 문제: 초보 채널에서 과도한 길이는 완주율 하락
- 해결: 3~8분 중심으로 테스트

### 7) 업로드 주제가 일관적인가?
- 문제: 주제가 자주 바뀌면 시청자층이 흔들림
- 해결: 시리즈 흐름 유지

### 8) 업로드 주기가 불규칙하지 않은가?
- 문제: 채널 기대 리듬이 깨짐
- 해결: 고정 요일/시간 유지

### 9) 댓글/반응을 다음 영상에 반영하고 있는가?
- 문제: 시청자 니즈 미반영
- 해결: 반복 질문을 다음 주제로 전환

### 10) 개선 없이 같은 방식 반복 중인가?
- 문제: 데이터 축적 대비 성장 정체
- 해결: 영상당 개선 1개 적용

---

## STEP 3) 정체 상황별 빠른 처방

### 왜 필요한가?
문제 유형마다 해결 포인트가 다릅니다.

### 실행 방법
아래 상황별로 대응하세요.

#### 상황 A: CTR 낮음
- 제목/썸네일 개선 우선
- 썸네일 문구 단순화

#### 상황 B: CTR은 괜찮은데 시청 지속 낮음
- 오프닝 단축
- 본문 핵심 앞배치

#### 상황 C: 시청은 괜찮은데 반응 약함
- 마무리 질문/CTA 추가
- 댓글 유도 문장 삽입

### 초보자 주의점
- 한 번에 여러 항목을 동시에 바꾸지 마세요.
- 원인 추적이 가능하도록 한 번에 1개만 수정하세요.

---

## STEP 4) 7일 개선 실험 루틴

### 왜 필요한가?
정체 구간은 감이 아니라 실험으로 풀어야 합니다.

### 실행 방법
1. 문제 1개 선택
2. 다음 영상에서 1개만 수정
3. 7일 데이터 비교
4. 효과 있으면 표준화

#### 복붙 실험 템플릿
```
[7일 개선 실험]
- 선택 문제:
- 수정 항목 1개:
- 수정 전 데이터:
- 수정 후 데이터:
- 유지/변경 결정:
```

### 초보자 주의점
- 1~2편만으로 최종 결론 내리지 마세요.
- 최소 3회 반복 실험 후 패턴을 보세요.

---

## STEP 5) 초보자 멘탈 관리 원칙

### 왜 필요한가?
정체 구간에서 멘탈이 흔들리면 운영이 끊깁니다.

### 실행 방법
아래 원칙을 지키세요.

1. 정체는 실패가 아니라 데이터 축적 구간
2. 비교 대상은 "대형 채널"이 아니라 "지난 내 영상"
3. 완벽주의 대신 반복 개선

### 초보자 주의점
- 조회수 급등 영상만 보고 방향을 급히 바꾸지 마세요.
- 시리즈의 일관성을 먼저 유지하는 것이 안전합니다.
