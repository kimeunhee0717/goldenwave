## **🎨 레이아웃 설계 상세 체크리스트 (7단계)**

### Step 1: 앨범 스펙 선정
- [ ] 사이즈 결정 (권장: 20cm x 20cm 정방형)
- [ ] 방향 선택: □ 가로형 □ 세로형 □ 정방형
- [ ] 페이지 수 확정: □ 20페이지 □ 30페이지 □ 40페이지
- [ ] 표지 재질 선택: □ 하드커버 □ 소프트커버 □ 스프링

### Step 2: 스토리보드 설계
- [ ] 페이지별 흐름 설계 (시작-전개-클리맥스-결말)
- [ ] 사진 배치 순서도 그리기 (종이에 스케치)
- [ ] 페이지당 사진 수 배분 계획:
  - 시작부(1-4p): 대표사진 중심
  - 전개부(5-15p): 다양한 구도
  - 결말부(16-20p): 감동적 마무리

### Step 3: 템플릿 선별 및 할당
- [ ] 템플릿 A (반반 - 2장) 할당할 페이지 체크
- [ ] 템플릿 B (1대3 배치) 할당할 페이지 체크
- [ ] 템플릿 C (폴리로이드) 할당할 페이지 체크
- [ ] 템플릿 D (4그리드) 할당할 페이지 체크
- [ ] 템플릿 E (풀페이지) 할당할 페이지 체크
- [ ] 그리드 시스템 적용 (골든비율/3분할)
- [ ] 여백 규칙 설정: 최소 1cm

### Step 4: 사진 크기 및 위치 배분
- [ ] 각 페이지당 메인 포토 1장 선정
- [ ] 서브 포토 배치 계획
- [ ] 크기 대비 조절 (60%:40% 또는 50%:50%)
- [ ] 시선 흐름 고려한 배치 (왼→오른, 위→아래)
- [ ] 페이지별 사진 배치 스케치 완료

### Step 5: 텍스트 공간 계획
- [ ] 캡션 공간 확보 (사진 아래 또는 옆)
- [ ] 날짜/장소/상황 기입 영역 확보
- [ ] 감성 문구 공간 확보
- [ ] 글꼴 크기 선정 (권장: 12-16pt)
- [ ] 텍스트 여백 미리 표시

### Step 6: 컬러 팔레트 및 배경
- [ ] 테마별 색상 스킴 정하기:
  - □ 파스텔톤 (모던)
  - □ 모노톤 (클래식)
  - □ 비비드 (활기찬)
  - □ 빈티지 (따뜻한)
- [ ] 배경 패턴/질감 선택
- [ ] 사진 톤과 배경 색 조화 확인

### Step 7: 디지털 목업 제작
- [ ] 캔바/포토샵 실행
- [ ] 실제 크기(20x20cm) 캔버스 생성
- [ ] 페이지별 시뮬레이션 제작
- [ ] 가족 피드백 수집
- [ ] 수정사항 반영

## **✍️ 스토리텔링 구성 체크리스트**

### □ 3막 구조 설정
- [ ] **1막 시작부** (페이지 1-4): 강렬한 첫인상
  - [ ] 표지: 가족 전체 사진
  - [ ] 페이지 1: 가장 감동적인 사진
  - [ ] 페이지 2-4: 다양한 순간 미리보기
  
- [ ] **2막 전개부** (페이지 5-15): 리듴감 있는 중간부
  - [ ] 시간/이벤트/감정 순서 결정
  - [ ] 큰 사진 ↔ 작은 사진 번갈아 배치
  - [ ] 에피소드별 구분
  
- [ ] **3막 결말부** (페이지 16-20): 감동적 마무리
  - [ ] 페이지 16-17: 가족 모두 함께하는 사진
  - [ ] 페이지 18: 가장 행복했던 순간
  - [ ] 페이지 19: 미래를 향한 사진
  - [ ] 페이지 20: 한 줄 메시지 + 손글씨

### □ 캡션 문구 선정
- [ ] 시작부용 문구 선택 (1-20번 중)
- [ ] 전개부용 문구 선택 (21-60번 중)
- [ ] 결말부용 문구 선택 (61-100번 중)
- [ ] 캡션 작성 (날짜+장소+감정 스토리)
