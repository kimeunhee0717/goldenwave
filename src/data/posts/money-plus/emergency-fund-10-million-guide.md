---

"비상금이 없어요"라고 말하는 사람들이 많습니다. 실제로 금융권 통계에 따르면 대한민국 직장인의 **40% 이상이 비상금 100만원도 없는 상태**입니다.

하지만 비상금 1000만원은 생각보다 어렵지 않습니다. **체계적인 계획**만 있으면 1년 안에 충분히 가능합니다.

오늘은 소득 구간별로 비상금 1000만원을 모으는 구체적인 로드맵을 공유합니다.

---

## 1. 비상금이 왜 중요할까?

### 1.1 비상금 없으면 어떤 일이?

- 갑작스러운 **의료비** → 신용대출 → 빚의 시작
- **실직** → 생활비 마련을 위해 고금리 대출
- **차 수리비** → 카드론 → 연 20% 이자

> **비상금은 보험입니다.** 위기 상황에서 빚을 안 지는 최후의 방어선입니다.

### 1.2 비상금은 얼마가 필요할까?

| 권장 기준 | 금액 |
|----------|------|
| 최소 기준 | **생활비 3개월** |
| 안전 기준 | **생활비 6개월** |
| 완벽 기준 | **생활비 12개월** |

월 생활비 200만원이라면, 최소 **600만원**, 안전하게는 **1200만원**이 비상금입니다.

## 2. 1년 1000만원 모으기: 소득별 로드맵

### 2.1 월급 200만원대

현실적으로 1년에 1000만원 모으기는 어렵습니다. 대신 **500~600만원**을 목표로 하세요.

| 구분 | 금액 |
|-----|------|
| 월 저축 목표 | 40~50만원 |
| 1년 저축액 | 480~600만원 |
| 핵심 전략 | 지출 최소화 + 부업 |

**실천법:**
- 월세 → 청약저축으로 이전 (월 10만원)
- 점심값 5천원 → 2천원으로 줄이기 (월 6만원 절약)
- 주말 알바 또는 부업 (월 20~30만원 추가)

### 2.2 월급 300만원대

이 구간부터 1년 1000만원이 현실적입니다.

| 구분 | 금액 |
|-----|------|
| 월 저축 목표 | 70~85만원 |
| 1년 저축액 | 840만원~1억 |
| 핵심 전략 | 자동이체 + 고정비 절감 |

**실천법:**
- 월급일 **자동이체** 70만원 설정
- 통신비: 알뜰폰 변경 (월 3만원 절약)
- 구독 서비스 정리 (월 2~3만원 절약)
- 카드 결제일 한 날짜로 통합

### 2.3 월급 400만원대 이상

이 구간은 1년 1000만원이 충분히 가능합니다.

| 구분 | 금액 |
|-----|------|
| 월 저축 목표 | 90~100만원 |
| 1년 저축액 | 1080만원~1200만원 |
| 핵심 전략 | 선저축 후지출 |

**실천법:**
- 월급일 바로 **100만원 자동이체**
- 남은 금액으로만 한 달 살기
- 보너스/성과금 100% 저축

## 3. 비상금 빨리 모으는 5가지 전략

### 3.1 선저축 후지출

돈이 남으면 저축하는 게 아닙니다. **먼저 저축하고 남은 돈으로 삽니다.**

```
잘못된 방식: 수입 - 지출 = 저축
올바른 방식: 수입 - 저축 = 지출
```

### 3.2 비상금 계좌 분리

급여 통장에 비상금을 두면 **무조건 씁니다.**

- 별도 CMA 또는 적금 계좌 개설
- 체크카드 없이 개설 (인출 불가능하게)
- 자동이체만으로 입금

### 3.3 52주 저축 챌린지

재미있게 저축하는 방법입니다.

| 주차 | 저축 금액 | 누적액 |
|-----|---------|--------|
| 1주 | 1만원 | 1만원 |
| 2주 | 2만원 | 3만원 |
| ... | ... | ... |
| 52주 | 52만원 | **1,378만원** |

주차별로 금액을 늘려가며 1년에 **1,378만원**을 모을 수 있습니다.

### 3.4 고정비 20% 줄이기

줄일 수 있는 고정비:

| 항목 | 절약 방법 | 월 절약액 |
|-----|---------|----------|
| 통신비 | 알뜰폰 변경 | 3만원 |
| 구독 서비스 | 안 쓰는 것 해지 | 2만원 |
| 보험료 | 다이렉트 보험 | 5만원 |
| 카드 연회비 | 무료카드로 변경 | 2만원 |
| **합계** | | **12만원** |

### 3.5 부업으로 가속화

본업 외 추가 수입:

| 부업 유형 | 월 평균 수입 |
|----------|------------|
| 주말 알바 | 30~50만원 |
| 블로그/유튜브 | 0~100만원 |
| 재능 판매 (과외, 번역) | 50~100만원 |
| 중고 물품 판매 | 10~30만원 |

## 4. 비상금 vs 투자: 무엇이 먼저?

### 4.1 정답은 **비상금 먼저**

비상금 없이 투자하면:

- 급한 돈 필요 시 → **펀드/주식 중도 해지**
- 하락장에 팔 수밖에 없음 → **큰 손실**

### 4.2 비상금 완성 후 투자

```
1단계: 비상금 3개월치 완성 (6~12개월)
2단계: 비상금 6개월치로 확대
3단계: 여유 자금으로 투자 시작
```
