# 2026-02-14 애드센스 재심사 작업리스트

## 목표
- AdSense 거절 사유(게시자 콘텐츠가 없는 화면 / 가치가 낮은 콘텐츠) 해소
- 재심사 시 "광고보다 본문 가치가 높은 사이트" 상태로 정비

## 범위
- 코드/콘텐츠 구조 점검 및 수정
- 저가치/빈화면/미완성 신호 제거
- 정책/신뢰 페이지 및 크롤링 신호 보강

## 작업 원칙
- 실제 수정 전 문제 URL/컴포넌트를 먼저 식별
- 사용자 경험과 크롤러 관점(콘텐츠 가시성)을 동시에 충족
- 임시/미완성 요소는 제거 또는 noindex 처리
- MD는 작성 포맷일 뿐이며, 최종 HTML에 실본문이 렌더링되는지로 판단

## MD/HTML 판단 기준
- [ ] MD 원문 보관 유지 (작성/관리 편의)
- [ ] 빌드 결과 HTML(`dist`)에 본문 텍스트가 실제 포함되는지 확인
- [ ] 주요 포스트 URL을 JS 비활성/크롤러 관점으로 점검
- [ ] "MD라서 미승인" 가정 금지, 실제 사유(빈 화면/저가치) 기준으로 수정

## 단계별 작업

### 1) 진단(우선)
- [x] 빈 콘텐츠 또는 얇은 콘텐츠 페이지 후보 수집
  - 라우트별 본문 길이/구성(제목, 본문, 근거, 작성자, 날짜) 점검
- [x] 미완성 UX 요소 점검
  - `href="#"` 링크, 동작하지 않는 폼/버튼, 임시 문구 탐지
- [x] 저가치 신호 점검
  - 자동 요약형/유사 반복형 글 비중 파악
  - 중복/유사 제목 및 유사 본문 패턴 점검

### 2) 구조/기술 수정
- [x] 빈/약한 페이지 정리
  - 본문 보강 가능 페이지는 보강
  - 불필요 페이지는 라우트 제거 또는 noindex 적용
- [x] 미완성 요소 제거
  - `href="#"` 링크를 실제 URL 또는 버튼 로직으로 교체
  - 문의 폼의 실제 전송 동작 연결(최소 mailto 또는 API 연동)
- [x] 크롤링/인덱싱 신호 정리
  - robots/sitemap/canonical/noindex 정책 점검
  - 공사중/관리자/내부용 경로 색인 제외 상태 재확인

### 3) 콘텐츠 가치 강화
- [ ] 핵심 글/페이지 품질 보강
  - 작성자, 발행일/수정일, 근거/출처, 실전 예시 추가
- [ ] 계산기/도구 페이지 본문 강화
  - 용도, 계산 로직 설명, 입력 예시, 주의사항 추가
- [ ] 자동 생성/요약 성격이 강한 글 정리
  - 중복·얕은 글 비공개 또는 통합
  - 사람이 작성한 분석형 콘텐츠 비중 확대

### 4) 검증
- [ ] 빌드 후 주요 URL 렌더링 확인
  - 홈/블로그/포스트/도구/정책 페이지 실본문 확인
- [ ] 포스트 샘플 10개 HTML 본문 검증
  - 제목, 리드문, 본문 H2/H3, 내부 링크, 작성자/날짜 확인
- [ ] 내부 링크 무결성 확인
  - 404/빈 링크/죽은 CTA 제거 확인
- [ ] AdSense 코드 및 ads.txt 유지 확인

### 5) 재심사 준비
- [ ] 변경사항 체크리스트 완료 여부 최종 확인
- [ ] 재심사 제출 전 대기 전략(크롤링 반영 3~7일)
- [ ] 재심사 제출 및 결과 추적 항목 정리
