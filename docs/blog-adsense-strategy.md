## 🎨 콘텐츠 품질 가이드라인

### 글 작성 템플릿

#### 구조적 요소 (필수)
```
1. 훅(Hook) - 첫 3문장으로 주목 끌기
2. 문제 인식 - 독자의 Pain Point 제시
3. 솔루션 소개 - 해결책 개요
4. 상세 방법 - 단계별 실행 가이드
5. 사례/예시 - 구체적 예시 제시
6. 핵심 요약 - Actionable Takeaways
7. CTA - 다음 행동 유도
```

#### 길이별 가이드

| 타입 | 단어 수 | 특징 | 예시 |
|------|---------|------|------|
| **스낵 콘텐츠** | 500-800 | 빠른 팁, 체크리스트 | "5분만에 하는 SEO 체크" |
| **기본 가이드** | 1,000-1,500 | 단일 주제 심층 | "키워드 리서치 완벽 가이드" |
| **권위 콘텐츠** | 2,000-3,000 | 종합 가이드, 리포트 | "2024 AI 마케팅 백서" |
| **핵심 콘텐츠** | 3,000+ | 궁극 가이드, 교과서 | "디지털 마케팅 완벽 바이블" |

#### SEO 최적화 체크리스트
- [ ] **타이틀**: 50-60자, 핵심 키워드 앞부분 배치
- [ ] **메타 설명**: 150-160자, 클릭 유도 문구 포함
- [ ] **헤딩 구조**: H1(1개) → H2 → H3 계층 구조
- [ ] **이미지**: alt 태그 포함, 1200px 이상 권장
- [ ] **내부 링크**: 관련 포스트 2-3개 연결
- [ ] **외부 링크**: 신뢰할 수 있는 출처 1-2개
- [ ] **키워드 밀도**: 1-2%, 자연스러운 배치
