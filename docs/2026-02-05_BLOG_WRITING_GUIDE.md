# 블로그 글 작성 가이드

## 개요

이 문서는 골든웨이브 인사이트 블로그에 새 글을 작성하고 게시하는 방법을 설명합니다.

---

## 파일 구조

블로그 시스템은 다음 파일들로 구성됩니다:

```
src/data/
├── posts.json          # 모든 포스트의 메타데이터
├── categories.json     # 카테고리 목록
├── authors.json        # 작성자 목록
└── posts/              # 마크다운 본문 파일
    ├── ai-marketing-guide-2024.md
    ├── seo-strategy-small-business.md
    └── [slug].md
```

---

## 블로그 글 작성 단계

### 1단계: 메타데이터 추가 (posts.json)

`src/data/posts.json` 파일을 열고 새 포스트 정보를 추가합니다.

```json
{
  "id": "5",
  "slug": "my-new-post",
  "title": "포스트 제목을 입력하세요",
  "excerpt": "포스트 요약을 150자 내외로 작성합니다. 블로그 목록에 표시됩니다.",
  "coverImage": "/images/blog/my-new-post.jpg",
  "categoryId": "marketing",
  "tags": ["태그1", "태그2", "태그3"],
  "authorId": "ceo",
  "publishedAt": "2024-02-05",
  "featured": false
}
```

#### 필드 설명

| 필드 | 설명 | 예시 |
|------|------|------|
| `id` | 고유 ID (순차적으로 부여) | `"5"` |
| `slug` | URL 경로 (영문, 하이픈만 사용) | `"my-new-post"` |
| `title` | 포스트 제목 | `"SEO 최적화 가이드"` |
| `excerpt` | 요약 (150자 내외) | `"검색 최적화 핵심 전략..."` |
| `coverImage` | 대표 이미지 경로 | `"/images/blog/seo.jpg"` |
| `categoryId` | 카테고리 ID | `"marketing"`, `"ai"`, `"seo"`, `"case-study"`, `"tips"` |
| `tags` | 태그 배열 (3-5개 권장) | `["SEO", "마케팅"]` |
| `authorId` | 작성자 ID | `"ceo"`, `"marketing-lead"`, `"tech-lead"` |
| `publishedAt` | 발행일 (YYYY-MM-DD) | `"2024-02-05"` |
| `featured` | 추천 글 여부 | `true` 또는 `false` |

---

### 2단계: 본문 작성 (Markdown 파일)

`src/data/posts/` 폴더에 `[slug].md` 파일을 생성합니다.

**파일명 규칙**: `posts.json`의 `slug` 값과 동일해야 합니다.
- slug가 `my-new-post`이면 파일명은 `my-new-post.md`

#### 마크다운 작성 예시

```markdown
서론 부분을 작성합니다. 독자의 관심을 끌 수 있도록 문제 제기나 흥미로운 통계로 시작하세요.

## 1. 첫 번째 섹션

본문 내용을 작성합니다. 명확하고 읽기 쉬운 문장으로 작성해주세요.

### 1.1 세부 항목

구체적인 내용을 다룹니다.

- **핵심 포인트 1**: 설명
- **핵심 포인트 2**: 설명
- **핵심 포인트 3**: 설명

## 2. 두 번째 섹션

### 2.1 실전 가이드

1. 첫 번째 단계: 설명
2. 두 번째 단계: 설명
3. 세 번째 단계: 설명

> 💡 **팁**: 독자에게 유용한 팁을 인용구로 강조합니다.

## 3. 세 번째 섹션

추가 내용을 작성합니다.

## 마무리

결론과 핵심 요약을 작성합니다. 독자가 다음 행동을 취할 수 있도록 CTA(Call to Action)를 포함하세요.
```

---

### 3단계: 이미지 추가

1. 대표 이미지를 `public/images/blog/` 폴더에 저장
2. 권장 크기: **1200x630px** (소셜 공유 최적화)
3. 파일 형식: **JPG** 또는 **WebP**
4. 파일명은 slug와 일치시키는 것을 권장

## 마크다운 작성 팁

### 제목 계층 구조

```markdown
## 대제목 (H2) - 주요 섹션
### 중제목 (H3) - 하위 섹션
#### 소제목 (H4) - 세부 항목 (필요시)
```

**주의**: H1(`#`)은 사용하지 않습니다. 포스트 제목이 H1로 렌더링됩니다.

### 텍스트 강조

```markdown
**굵은 글씨** - 핵심 키워드 강조
*기울임* - 용어, 인용
`코드` - 기술 용어, 명령어
~~취소선~~ - 잘못된 정보 표시
```

### 목록

```markdown
순서 없는 목록:
- 항목 1
- 항목 2
  - 중첩 항목

순서 있는 목록:
1. 첫 번째
2. 두 번째
3. 세 번째
```

### 인용구 (팁, 주의사항)

```markdown
> 일반 인용구

> 💡 **팁**: 유용한 정보

> ⚠️ **주의**: 주의사항

> 📌 **참고**: 참고 정보
```

### 표

```markdown
| 열1 | 열2 | 열3 |
|-----|-----|-----|
| 내용 | 내용 | 내용 |
| 내용 | 내용 | 내용 |
```

### 링크

```markdown
[링크 텍스트](https://example.com)
[내부 링크](/blog/other-post)
```

### 이미지 (본문 내)

```markdown
![이미지 설명](/images/blog/example.jpg)
```

## 글 작성 체크리스트

### 작성 전
- [ ] 주제와 타겟 독자 정의
- [ ] 키워드 리서치 완료
- [ ] 개요 작성

### 작성 중
- [ ] 명확한 제목 (50자 이내)
- [ ] 핵심을 담은 요약 (150자 이내)
- [ ] 본문에 H2, H3 제목 적절히 사용
- [ ] 이미지/표/목록으로 가독성 향상
- [ ] CTA(Call to Action) 포함

### 작성 후
- [ ] 맞춤법 검사
- [ ] 링크 동작 확인
- [ ] 이미지 표시 확인
- [ ] posts.json에 메타데이터 추가
- [ ] 대표 이미지 업로드

## SEO 최적화 가이드

### 제목 (title)
- 50자 이내
- 핵심 키워드를 앞에 배치
- 숫자나 연도 포함시 클릭률 상승

**좋은 예**: `2024 SEO 완벽 가이드: 검색 1페이지 달성하기`
**나쁜 예**: `SEO에 대해서 알아보기`

### 요약 (excerpt)
- 150자 내외
- 글의 핵심 가치를 명확히 전달
- 독자의 문제/니즈 언급

### 태그 (tags)
- 3-5개 권장
- 검색 가능한 키워드 선택
- 너무 광범위하거나 너무 구체적인 태그 피하기

## 실제 작성 예시

### posts.json 추가 내용

```json
{
  "id": "5",
  "slug": "instagram-marketing-2024",
  "title": "2024 인스타그램 마케팅 완전 정복: 릴스부터 협업까지",
  "excerpt": "인스타그램 알고리즘 변화에 맞춘 최신 마케팅 전략을 알아봅니다. 릴스 활용법, 인플루언서 협업, 쇼핑 기능까지 실전 가이드를 제공합니다.",
  "coverImage": "/images/blog/instagram-marketing-2024.jpg",
  "categoryId": "marketing",
  "tags": ["인스타그램", "SNS 마케팅", "릴스", "인플루언서", "2024 트렌드"],
  "authorId": "marketing-lead",
  "publishedAt": "2024-02-05",
  "featured": true
}
```

### instagram-marketing-2024.md 본문

```markdown
2024년 인스타그램은 그 어느 때보다 강력한 마케팅 채널로 자리잡았습니다. 20억 명 이상의 월간 활성 사용자를 보유한 이 플랫폼에서 효과적으로 마케팅하는 방법을 알아보겠습니다.

## 1. 2024 인스타그램 알고리즘 이해하기

인스타그램 알고리즘은 지속적으로 변화하고 있습니다. 현재 알고리즘이 중요하게 보는 요소들을 살펴보겠습니다.

### 1.1 핵심 랭킹 요소

- **참여도**: 좋아요, 댓글, 공유, 저장
- **관계성**: 사용자와의 상호작용 이력
- **적시성**: 최신 콘텐츠 우선
- **체류 시간**: 콘텐츠에 머무는 시간

> 💡 **팁**: 게시 후 30분 내 참여도가 알고리즘 노출에 가장 큰 영향을 미칩니다.

## 2. 릴스(Reels) 전략

릴스는 현재 인스타그램에서 가장 높은 도달률을 보이는 콘텐츠 형식입니다.

### 2.1 릴스 제작 가이드

1. **첫 3초가 핵심**: 시선을 사로잡는 도입부
2. **최적 길이**: 15-30초 (짧고 임팩트 있게)
3. **트렌드 활용**: 인기 오디오와 챌린지 참여
4. **텍스트 추가**: 음소거 시청자를 위한 자막

## 3. 인플루언서 협업

효과적인 인플루언서 협업을 위한 전략을 알아봅니다.

| 인플루언서 유형 | 팔로워 수 | 장점 |
|---------------|----------|------|
| 나노 | 1K-10K | 높은 참여율, 신뢰도 |
| 마이크로 | 10K-100K | 균형 잡힌 도달과 참여 |
| 매크로 | 100K-1M | 넓은 도달 범위 |
| 메가 | 1M+ | 브랜드 인지도 향상 |

## 마무리

인스타그램 마케팅 성공의 핵심은 일관성과 가치 제공입니다. 오늘 소개한 전략들을 꾸준히 실행하면 분명 성과를 볼 수 있을 것입니다.

**인스타그램 마케팅 전략 수립이 필요하시다면 [문의하기](/contact)를 통해 연락주세요.**
```

## 자주 하는 실수

1. **slug 불일치**: posts.json의 slug와 마크다운 파일명이 다름
2. **이미지 경로 오류**: `/public` 제외하고 `/images/...`로 작성
3. **날짜 형식 오류**: `YYYY-MM-DD` 형식 준수
4. **H1 사용**: 본문에서 `#` 사용 금지 (제목은 자동 생성)
5. **카테고리/작성자 ID 오타**: 정의된 ID 정확히 사용
